# simple-calculator
## Features:-
- explicit fields for `first number` and `second number` to ensure clean data entry.
- supports addition, subreaction , multiplication , and division.
- a clear button to quickly wipe out inputs and reset the calculator.

  ## concepts applied:-
  - targeting muliple inputs using `document.getElementById` .
  - converting input string values into numbers (Number() or parseFloat()) before executing math operations.
  - -using click event.
